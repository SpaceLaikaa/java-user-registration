#  Java User Registration Module

The backend module provides support for handling user registration logic . This project demonstrates Test Driven Development   using **Java** and JUnit 5 Automation Testing**.

Overview

This project develops the `UserService` class, which validates the user input before registration. The key focus is on defensive coding to ensure that any invalid data like null values, short passwords, or underage users-do not enter the system.

- Automated Testing with JUnit 5

Tech Stack
* **Language:** Java JDK 21
* **Testing:** JUnit 5
* **Build Tool:** Maven
* **IDE:** IntelliJ IDEA
